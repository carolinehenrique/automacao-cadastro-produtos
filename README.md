# 🤖 Automação de Cadastro em Larga Escala

## 📌 Sobre o projeto
Este projeto consiste na criação de uma solução em Python para automação do cadastro massivo de produtos em sistema interno.

O processo, que antes era manual e demorado, passou a ser executado automaticamente, podendo rodar de forma programada ou sob demanda.

## 🚀 Tecnologias utilizadas
- Python
- RPA
- Web Scraping

## 💡 Resultados
- Redução significativa de tempo operacional
- Automação de tarefas repetitivas
- Aumento de eficiência no cadastro de produtos
